Technique Used:
This image was created to be used with the animated banner on the client's website. Originally the image was on a background, I cut this out using the Pen tool, and then creating a path and using the selection tool to remove the areas of the image I did not want to keep. I made the background transparent and saved as a PNG (through save for web), I chose PNG because this file format is lossless and allows 24bit colour, it also allows transparency, however I am aware of some potential browser issues.
